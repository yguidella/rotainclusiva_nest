#### 🎯 O Projeto
O **Rota Inclusiva** é uma solução de **Back-end** desenvolvida para enfrentar os desafios de mobilidade urbana de pessoas com deficiência ou mobilidade reduzida. Através de uma API REST, conectamos motoristas voluntários e passageiros em uma rede de caronas colaborativas e acessíveis.

---

#### 🚀 Funcionalidades Principais
* **Gestão de Perfis:** Cadastro completo de usuários e motoristas com foco em necessidades de acessibilidade.
* **Sistema de Caronas:** Endpoints estruturados para oferta, listagem e gerenciamento de viagens.
* **Cálculo de Previsibilidade:** Lógica especializada para converter distância e velocidade em tempo real (Horas/Minutos).
* **Integridade de Dados:** Relacionamento robusto entre tabelas de usuários, categorias e viagens via TypeORM.

---

#### 🛠️ Stack Técnica
* **Framework:** NestJS (Node.js)
* **Linguagem:** TypeScript
* **Banco de Dados:** MySQL
* **Versionamento:** Git & GitHub
* **Documentação/Testes:** Insomnia

---

#### 🤝 Fluxo Colaborativo
* **Metodologia Ágil:** Organização de tarefas e prazos focada em entregas contínuas.
* **Gestão de Código:** Uso de branches e Pull Requests para integração segura entre os desenvolvedores.
* **Comunicação Integrada:** Alinhamento técnico via Discord e suporte visual via Canva.

---

#### 🌍 Impacto Social
O diferencial do **Rota Inclusiva** é transformar dados técnicos em **autonomia**. O cálculo de tempo estimado não é apenas um número, mas uma ferramenta de planejamento essencial para garantir que o usuário PcD tenha controle total sobre seu deslocamento, minimizando imprevistos em trajetos não adaptados.
